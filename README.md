# Elite Train iPhone

Version mobile statique et installable de l'app.

## Ce qu'il y a dedans

- `index.html` : point d'entree
- `styles.css` : interface mobile iPhone
- `app.js` : logique complete
- `manifest.webmanifest` : installation type PWA
- `service-worker.js` : cache offline basique

## Pour l'avoir sur iPhone

1. Heberge le dossier `elite-train-iphone` sur un site HTTPS statique.
2. Ouvre l'URL dans Safari sur l'iPhone.
3. Touche `Partager`.
4. Choisis `Ajouter a l'ecran d'accueil`.

## Options simples d'hebergement

- Glisser le dossier sur un hebergeur statique.
- Le pousser sur GitHub Pages.
- Le mettre sur n'importe quel serveur HTTPS de fichiers statiques.
